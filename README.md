

# Time Frame SaveGuard

a javascript injection to stop you from clicking the "Save" button if you have an uncommitted date/time in a timeframe modal

### to launch

```bash
git clone git@github.com:DallanL/saveguard.git
cd saveguard
```

update env variables
```bash
cp .env.example .env
vim .env
```

deploy to docker
```bash
docker compose up -d
```
